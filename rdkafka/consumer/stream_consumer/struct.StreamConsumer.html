#[must_use = "Consumer polling thread will stop immediately if unused"]pub struct StreamConsumer<C: ConsumerContext + 'static = DefaultConsumerContext> { /* fields omitted */ }

A Kafka consumer providing a futures::Stream interface.

This consumer doesn't need to be polled explicitly since awaiting the stream returned by StreamConsumer::start will implicitly poll the consumer.
